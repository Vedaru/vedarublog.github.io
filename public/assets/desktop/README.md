# Desktop Wallpaper

此文件夹用于存放桌面隐喻风格的壁纸资源。

## 使用说明

请在此文件夹中放置名为 `wallpaper-default.jpg` 的壁纸图片，该图片将作为默认桌面背景。

### 推荐规格

- **格式**: JPG, PNG, WEBP
- **尺寸**: 1920x1080 或更高
- **文件大小**: < 500KB（优化后）
- **风格**: 适合长时间观看的低对比度背景，避免过于鲜艳

### 示例资源

你可以从以下网站获取免费壁纸：
- https://unsplash.com/
- https://www.pexels.com/
- https://pixabay.com/

### 替代方案

如果不想使用壁纸，可以在 `src/styles/desktop-metaphor.css` 中修改 `.desktop-container` 的背景颜色或渐变。

当前配置中使用的是青色渐变背景（#008080）作为后备方案。
